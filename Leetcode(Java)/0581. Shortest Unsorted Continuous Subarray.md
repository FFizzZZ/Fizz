## Sort
```
class Solution {
    public int findUnsortedSubarray(int[] nums) {
        int[] num = nums.clone();
        Arrays.sort(num);
        int l = num.length, r = 0;
        for (int i = 0; i < num.length; i++)
            if (num[i] != nums[i]) {
                l = Math.min(i, l);
                r = Math.max(i, r);
            }
        return r - l >= 0 ? r - l + 1 : 0;
    }
}
```
