## Prefix Sum
```
class Solution {
    public int minFlipsMonoIncr(String S) {
        int n = S.length();
        int[] prefix = new int[n + 1];
        for (int i = 0; i < n; i++)
            prefix[i + 1] = prefix[i] + (S.charAt(i) == '1' ? 1 : 0);
        int ans = Integer.MAX_VALUE;
        for (int i = 0; i <= n; i++)
            ans = Math.min(ans, prefix[i] + n - i - (prefix[n] - prefix[i]));
        return ans;
    }
}
```
