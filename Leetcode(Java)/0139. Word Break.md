## Dynamic Programming
```
class Solution {
    public boolean wordBreak(String s, List<String> wordDict) {
        boolean[] dp = new boolean[s.length() + 1];
        dp[0] = true;
        for (int i = 0; i <= s.length(); i++) {
            for (String word : wordDict) {
                int l = word.length();
                if (dp[i] && i + l <= s.length() && s.substring(i, i + l).equals(word))
                    dp[i + l] = true;
            }
        }
        return dp[s.length()];
    }
}
```
