## HashSet and Intelligent Sequence Building
```
class Solution {
    public int longestConsecutive(int[] nums) {
        Set<Integer> set = new HashSet<>();
        for (int n : nums)
            set.add(n);
        int ans = 0;
        for (int n : nums) {
            if (!set.contains(n - 1)) {
                int cur = n + 1;
                while (set.contains(cur)) cur++;
                ans = Math.max(ans, cur - n);
            }
        }
        return ans;
    }
}
```
