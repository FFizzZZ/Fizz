#### No Duplication
```
class Solution:
    def minNumberInRotateArray(self, rotateArray):
        l, r = 0, len(rotateArray) - 1
        right = rotateArray[-1]
        while l < r:
            mid = (l + r) / 2
            val = rotateArray[mid]
            if val <= right:
                r = mid
            else:
                l = mid + 1
        return rotateArray[l]
```
