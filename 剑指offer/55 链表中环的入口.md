## 双指针-剑指offer
```
class Solution:
    def EntryNodeOfLoop(self, pHead):
        p1, p2 = pHead, pHead
        flag = 0
        while p1.next:
            p1, p2 = p1.next.next, p2.next
            if p1 == p2:
                flag = 1
                break
        if not flag: return None
        else:
            p1, count = p1.next, 1
            while p1 != p2:
                p1, count = p1.next, count + 1
            p1, p2 = pHead, pHead
            for i in range(count): p1 = p1.next
            while p1 != p2:
                p1, p2 = p1.next, p2.next
            return p1
```

## 断链
只在保证有环的情况下可以用。破坏了环，实际上没什么意义。
```
class Solution:
    def EntryNodeOfLoop(self, pHead):
        if not pHead.next: return None
        p, previous = pHead.next, pHead
        while p:
            previous.next = None
            previous = p
            p = p.next
        return previous
```
