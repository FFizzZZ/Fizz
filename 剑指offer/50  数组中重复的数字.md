        d = {}

## 遍历一个每个数字，在数组相应位置加上n
```
class Solution:
    def duplicate(self, numbers, duplication):
        n = len(numbers)
        for i, v in enumerate(numbers):
            if v > n - 1: v -= n
            if numbers[v] > n - 1:
                duplication[0] = v
                return True
            else:
                numbers[v] += n
        return False
```
可能存在溢出的问题，所以可以采用减去n
```
class Solution:
    def duplicate(self, numbers, duplication):
        n = len(numbers)
        for i, v in enumerate(numbers):
            if v < 0: v += n
            if numbers[v] < 0:
                duplication[0] = v
                return True
            else:
                numbers[v] -= n
        return False
```
