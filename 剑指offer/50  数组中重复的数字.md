## 交换 numbers[i] 和 numbers[[number[i]]
```
class Solution:
    def duplicate(self, numbers, duplication):
        i = 0
        while i < len(numbers):
            val = numbers[i]
            if i == val:
                i += 1
            else:
                if numbers[val] == val:
                    duplication[0] = val
                    return True
                else:
                    numbers[i], numbers[val] = numbers[val], val
        return False
```
## 遍历一个每个数字，在数组相应位置加上n
```
class Solution:
    def duplicate(self, numbers, duplication):
        n = len(numbers)
        for i, v in enumerate(numbers):
            if v > n - 1: v -= n
            if numbers[v] > n - 1:
                duplication[0] = v
                return True
            else:
                numbers[v] += n
        return False
```
可能存在溢出的问题，所以可以采用减去n
```
class Solution:
    def duplicate(self, numbers, duplication):
        n = len(numbers)
        for i, v in enumerate(numbers):
            if v < 0: v += n
            if numbers[v] < 0:
                duplication[0] = v
                return True
            else:
                numbers[v] -= n
        return False
```

## 在不修改数组的情况下
可以将数组分成两份，1/~m, m+1/~n。
若1~m的组中有超过m个数，那么重复数字在1~m中，然后一直二分下去。
缺点是不能找出所有重复数字。
