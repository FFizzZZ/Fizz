## 翻转两次
```
class Solution:
    def ReverseSentence(self, s):
        # write code here
        s = s[::-1]
        ans = ''
        temp = ''
        for i in s:
            if i != ' ':
                temp += i
            else:
                ans += temp[::-1] + ' '
                temp = ''
        ans += temp[::-1]
        return ans
```

## 用栈存储单词，然后出栈。
```
class Solution:
    def ReverseSentence(self, s):
        # write code here
        ans = []
        temp = ''
        for i in s:
            if i != ' ':
                temp += i
            else:
                ans.append(temp)
                temp = ''
        ans.append(temp)
        return ' '.join(ans[::-1])
```


## 内置函数
```
class Solution:
    def ReverseSentence(self, s):
        return " ".join(s.split()[::-1]) if s.strip() != "" else s
```
