```
class Solution:
    def jumpFloor(self, number):
        # write code here
        if number == 1: return 1
        if number == 2: return 2
        dp = [0] * number
        dp[0], dp[1] = 1, 2
        for i in range(2, number):
            dp[i] = dp[i-1] + dp[i-2]
        return dp[-1]
```
其实这是个斐波那契数列。
