```
class Solution:
    def jumpFloorII(self, number):
        # write code here
        if number == 1: return 1
        if number == 2: return 2
        dp = [0] * number
        dp[0], dp[1] = 1, 2
        for i in range(2, number):
            dp[i] = sum(dp) + 1
        return dp[-1] 
```
可以证明 $f(n)=2^(n-1)$.
