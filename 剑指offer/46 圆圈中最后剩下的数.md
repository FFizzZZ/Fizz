```
class Solution:
    def LastRemaining_Solution(self, n, m):
        if n == 0: return -1
        array = list(range(n))
        idx = 0
        for _ in range(len(array) - 1):
            idx = (idx + m - 1) % len(array)
            array.pop(idx)
        return array[0]
```

## 数学推导
#### F(n, m) = [F(n - 1, m) + m] % n
```
class Solution:
    def LastRemaining_Solution(self, n, m):
        if n == 0: return -1
        idx = 0
        for i in range(1, n + 1):
            idx = (idx + m) % i
        return idx
```
